Mit tud egy szoftverfejlesztő és -tesztelő technikus?

Széleskörű kompetenciák a modern szoftverfejlesztés területén

A szoftverfejlesztő és tesztelő technikus tanulmányai során az alábbi területeken szerez széleskörű ismereteket és készségeket:

  • Verziókezelés és csapatmunka

    Verziókezelő-rendszerek (Git, Github) magabiztos használata és csapatmunka eszközök használata (Slack, Trello, Microsoft Teams)

  • Python programozás

    Szkriptek és programok készítése Pythonban, akár objektumorientált megközelésítésben is

  • Hardver és operációs rendszerek

    Hardvereszközök és operációs rendszerek magabiztos kezelése (Windows, Linux, mobil operációs rendszerek)

  • Hálózati ismeretek

    Kis- és közepes hálózatokban jól használható hálózati ismeretek és készségek (IP címzés, protokollok, WiFi router konfiguráció), hálózatbiztonsági beállítások

  • Virtualizáció és felhőtechnológiák

    Alapvető virtualizációs technikák használata (virtuális gépek, Docker), felhőszolgáltatások használata

  • Frontend fejlesztés

    Összetett webes frontend fejlesztési feladatok elvégzése HTML5, CSS3 és egy választott JavaScript keretrendszer vagy könyvtár használatával (pl. React, Vue, Angular)

  • Backend fejlesztés

    Backend fejlesztés Node.js-ben vagy PHP-ban megfelelő keretrendszerek vagy könyvtárak használatával (pl. Express, Laravel, Spring Boot)

  • REST API fejlesztés

    REST API fejlesztése és dokumentálása

  • OOP fejlesztés (C#/Java)

    Asztali és általános célú objektumorientált szoftverek fejlesztése C# vagy Java nyelvi környezetben haladó szinten, objektumorientált programozás alkalmazása

  • Tiszta kód elvei

    Tiszta kód elveinek alkalmazása és kódminőség biztosítása

  • Adatbáziskezelés

    SQL adatbázisok tervezése, létrehozása és magabiztos kezelése, ORM technológiák használata szoftverfejlesztésben

  • Mobil alkalmazásfejlesztés

    Mobil alkalmazások fejlesztése Android eszközökre (pl. Native Java Android, React Native, Progressive Web App)

  • CMS rendszerek

    Tartalomkezelő rendszerek (CMS, pl. WordPress) használata, testreszabása és egyedi bővítmények fejlesztése

  • Szoftvertesztelés

    Unit tesztelés és integrációs tesztelés elvégzése

  • Szakmai kommunikáció

    Szakmai angol kommunikáció (KER B1 szint) és műszaki dokumentáció készítése

Három év tapasztalata alapján azt látom, hogy azok a diákok, akik duális képzésben nálunk tanulnak képesek hasznos tagjaivá válni a szakmai csapatunknak. Képesek team munkában dolgozni, önálló projekteket végig vinni, kisebb projektek esetén önállóan megoldani. Nem csak szakmailag fejlődtek, megismerték a céges kultúrát. Nemrégiben választaniuk kellett az osztálykirándulás és a céges csapatépítő között, mert egy időben szerveződött a két program. A céges csapatépítő mellet döntöttek.
